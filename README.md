# KGU dormitory cafeteria menu application
🍱 경기대학교 기숙사 식당 식단표 어플리케이션
### Description
텅장이 된 지금, 기숙사 식당을 애용하려고 한다.
기숙사 식단표를 보기 위해서는 홈페이지나 에타에서 찾아야 하는 번거로움에 간단한 심심풀이로 이번주 식단을 알려주는 어플리케이션을 만들어봤다.
### Environment
- Language : Java
- Framework : android studio
